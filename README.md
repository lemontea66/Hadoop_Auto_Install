# Hadoop_Auto_Install
 CentOS下Hadoop环境一键安装脚本

![][1]

## 所需系统环境 ##

 1. 纯净CentOS 6.X 32位

> 注意：系统需要root用户运行
> 请务必参照文章[https://www.shujuhe.cn/archives/44/][2]配套完成安装

## 使用方法 ##
命令行内运行下段代码即可

    wget -q http://file.shujuhe.cn/master.sh && sh master.sh
